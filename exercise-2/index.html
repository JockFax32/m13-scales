Welcome to Scales: Exercise 2

This exercise challenges you to bring together your D3 skills for defining scales, building charts, and positioning axes. In your js/main.js file, do the following:

  • Read in your data/prepped_data.csv file using the d3.csv function
  • Define scales to position GDP in a horizontal direction, and Life Expectancy in a vertical direction. You should use log and linear scales (respectively)
  • Add circle elements to your screen for each element in your dataset by performing a data-join. Note, you should bind data to the g element insdie your SVG to better facilitate arranging your elements
  • Define axes and append them to your DOM
  • Append text elements to your DOM to represent axis labels.
  • Bonus: Color each point by region, and add a legend (remember, a legend is just a set of HTML elements that look like a legend)
  • Bonus: Size each point by fertility, and add a legend (remember, a legend is just a set of HTML elements that look like a legend)
